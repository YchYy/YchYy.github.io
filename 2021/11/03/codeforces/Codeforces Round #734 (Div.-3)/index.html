# Codeforces Round #734 (Div. 3)

# C. Interesting Story

# 题意

给你几个字符串 (仅由 a, b, c, d, e 五个字母构成)，让你找到最大的字符串个数， 满足这些字符串中的一个字母的个数大于其他字母的个数和。

# 思路

贪心，记一下字符串某个字母个数 * 2 与该字符串的长度之差排序，看最多多少个字符串后该字母个数 ×2 小于等于字符串总长度。

# 题意

给你 n × m 的网格 (偶数)，k 个水平的 1 × 2 的骨牌，问用这 k 个水平的 1 × 2 的骨牌和无数竖直的 2 × 1 的骨牌能否填满整个网格。

# 思路

分类讨论

  1. 当 n 和 m 均为偶数时，只需 k % 2 == 0 即可；
  2. 当 n 为奇数，m 为偶数时，先用 m/2 个填满一行，转化为情况一；
  3. n 为偶数，m 为奇数同二。
