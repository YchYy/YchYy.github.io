# Codeforces Round #742 (Div. 2)

# C. Carrying Conundrum

# 题意

爱丽丝的加法学错了， 正确的进位是进到前一位中， 而爱丽丝的是进在前两位中， 给出爱丽丝算出的答案 n ， 问她有多少中方案 (a, b) 能算出 n. a, b 为正整数.

正确的

爱丽丝的

# 思路

由于爱丽丝的加法为进两位，所以奇数位与偶数互不影响.

如 15005 奇数位为 105, 偶数位为 50, 构成奇数位的方案数有 106 种，偶数位有 51 种，故总方案数有 106 * 51 种，又因为 a, b 为正整数，所以要再减去 a 为零与 b 为零的情况。所以 总方案数 = (奇数位 + 1) * (偶数位 + 1) - 2;

# D. Expression Evaluation Error

# 题意

求 n 个数，这 n 个数的和十进制下为 s , 问 这 n 个数是多少时，其 十一进制 下的和最大.

# 思路

由于十进制是逢十进一，而十一进制是十一进一，所以当这 n 个数相加每进一位其十一进制下的和会少一.

所以当这 n 个数不产生进位的时候其 十一进制 下的和最大

一个数最多由多少个数相加而不产生进位？ 例如 4523 显然 为 4 + 5 + 2 + 3, 最多由这个数的各个位置之和相加而不产生进位.

若产生进位，则要让其产生进位的次数更少.
