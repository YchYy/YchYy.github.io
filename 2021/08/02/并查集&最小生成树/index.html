# 并查集

并查集（Union-find Sets）是一种非常精巧而实用的数据结构，它主要用于处理一些不相交集合的合并问题

# 基本操作

​ 合并：把两个集合合并成一个大集合

​ 查询：查询一个元素属于哪一个集合

# 具体步骤

1. 并查集的储存

​ 使用一个数组 f 保存父节点（根的父节点设为自己）

2. 并查集的初始化

​ 设有 n 个节点，刚开始时每个节点的父节点均为自己

3. 并查集的查询操作

​ 若 x 是树根，则 x 就是集合代表，否则递归访问 f [x] 直至根节点

4. 并查集的合并操作

​ 合并元素 x 和元素 y 所在的集合，等价于让 x 的树根作为 y 的树根的子节点
