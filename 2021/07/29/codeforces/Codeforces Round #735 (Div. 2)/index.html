# Codeforces Round #735 (Div. 2)

# A.Cherry

​

# B.Cobb

# 题意

给定 n 和 k， n 个数 a1_11​, a2_22​, ... , an_nn​. 求 i * j - k * (ai_ii​ | aj_jj​) 的最大值 (1 ≤\leq≤ i < j ≤\leq≤ n).

# 分析

当 n ≤\leq≤ 500(任取别太小别 T 就行) 时暴力， 当 n 太大时， 由于 1 ≤\leq≤ k ≤\leq≤ min (100, n), 所以 i * j 价值更高， 所以取后面的数得到的结果更大，只需取后 500 个数即可。

# C.Mikasa

# 题意

给定一个 n 和 m 求一个最小的数 ans 使 ans \notint {n xor 0, n xor 1, ......, n xor m}.

# 分析

求 n xor i ≠\neq= ans, 可转化为 n xor ans ≠\neq= i. 又因为 i ∈\in∈ {0, 1, 2, ... , m}, 则问题可转化为 n xor ans > m 即 n xor ans ≥\geq≥ m + 1.

x xor y = z 等价于 x xor z = y
