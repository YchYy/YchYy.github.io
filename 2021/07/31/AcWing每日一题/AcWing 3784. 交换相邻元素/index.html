# AcWing 3784. 交换相邻元素

# 题意

​ 给长度为 n 的数组该数组是 1 ~ n 的排序，和一个长度为n−1n-1n−1 的 01 字符串，第iii 个字符为 1 表示第 i 位可以与第 i + 1 位进行交换操作，第 i 个字符为 0 表示第 i 个位置无法进行交换操作。

​ 问：是否可以将这 n 个数通过交换操作变成升序序列。能则输出 YES，否则输出 NO。

​

# 分析

​ 假设 [l, r] 的区间内字符全为 1，这 [l , r + 1] 这段区间的数必定能按升序排序，由于是 1 ~ n 的数组，则 [l, r + 1] 内的最大值不能超过 r + 1, 最小值不能低于 l，则只需要判断字符为 0 的位置 i 的数是否为 i (除了 1 后面的 那个 0)，全为 1 的区间 [l , r]，[l, r + 1] 是否在 l ~ r + 1 之间即可 (由于我们从 1 ~ n 遍历，其实只要判断最大值是否大于 r + 1 即可)。

E=∑i=1np(i)∗w(i)E=\displaystyle\sum_{i=1}^np(i)*w(i)E=i=1∑n​p(i)∗w(i)。
